about me

Husband, father of 4, software developer, code perfectionist. I enjoy spending time with my family, cooking, playing video games, riding bikies, playing guitar and writing code (of course). My favorite color is purple (obviously).

I've been developing software since 1995. My first gig was writing database apps in Borland Paradox. While at Eureka eSolutions I was introduced to ASP (classic). While this seemed easier than the CGI I was introduced while persuing my Bachelor of Arts in Computer Science at The University of Texas at Austin in 1994, scripting at the time felt like playing with Tinker Toys. When ASP.NET came along it seemed an improvement over VBScript. My only problem with WebForms was that it tried to do too much for you as a developer. I wanted to get back to writing the HTML myself. Finally ASP.NET MVC came along and brought both worlds together. Nowadays the web is moving toward putting as much UI rendering and business logic as possible in the browser using JavaScript frameworks like Angular and only using the server as a place to get and store data. So nowadays I focus my attention knowing the fundamentals of HTML, CSS and JavaScript. Something I've become very adept at as a student mentor for Thinkful.com.
